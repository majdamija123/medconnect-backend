# MedConnect Backend

API backend pour l'application MedConnect développée avec Django REST Framework.

## Technologies
- Django 4.2+
- Django REST Framework
- PostgreSQL
- JWT Authentication (à venir)

## Installation

### 1. Cloner le projet
```bash
git clone https://github.com/majdamija123/medconnect-backend.git
cd medconnect-backend

cd C:\Users\pc\Desktop\medconnect_app\medconnect-backend
my_venv\Scripts\activate
python manage.py runserver 0.0.0.0:8000

python manage.py migrate
